# 🎯 Guess My Number Game

A simple and interactive number guessing game built using **HTML, CSS, and JavaScript**. The objective is to guess a randomly generated number between 1 and 20 within the fewest attempts possible while maintaining the highest score.


---

## 📌 Features

- 🎲 Random number generated between 1 and 20
- 📊 Score system that decreases with wrong guesses
- 🏆 High score tracking
- 💬 Instant feedback messages (Too high / Too low / Correct)
- 🔄 Restart game functionality
- 🎨 Dynamic UI changes (color, size, messages)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

---

## 🎮 How to Play

1. Open the game in your browser.
2. Enter a number between **1 and 20**.
3. Click **Check!** to submit your guess.
4. Follow the hints:
   - 📈 Too high → Try a smaller number
   - 📉 Too low → Try a bigger number
5. Try to guess correctly with the highest score possible.
6. Click **Again!** to restart the game.
